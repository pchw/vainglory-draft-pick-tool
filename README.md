# vainglory-draft-pick-tool

This is Vainglory Draft Pick Tool!
You can use this on private match or some tournaments when broadcasting.

これはベイングローリーのドラフトピックツールです．
プラベとか大会を配信する際に視聴者に分かりやすく何を選ばれているかを見せるやつです．
